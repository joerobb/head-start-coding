Intro Course

Beginner HTML and CSS

If you want to create incredible websites and apps, you'll need two key programming languages: HTML and CSS. This course covers the basics of both languages and introduces you to using AI for coding assistance.

Flip the tiles below to see the skills you will gain by the end of the course.

HTML

The building blocks of all web pages.

HyperText Markup Language

Headings, paragraphs, images, forms and links - HTML is the code that is used to build a web page and it's content. In this course you will learn how to create all of the elements needed to structure a basic web page.

Flip
CSS

Make your website beautiful.

Cascading Style Sheets

CSS is used for styling and laying out webpages. It is used to adjust content size, spacing, colour and font. In this course you will learn how to style and lay out a web page to make it look fantastic.

AI Guidance

AI is an essential new skill for new developers. Here you will learn how to use it to boost your coding, automate routine tasks, and troubleshoot issues.

Frequently Asked Questions

FAQs

What will I need for the course?

Ideally two devices such as a laptop and an iPad to allow you to follow the tutor on one screen and write your code on the other. However, one large screen split down the middle will also be fine. We will help you get everything set up for the course.

I'm a parent, how can I track my child's progress?

Our courses are "project based" meaning that your child will develop a portfolio of accomplishments to demonstrate their progress.

What if I miss a lesson?

Lesson recordings can be provided if a student was to miss a lesson.
